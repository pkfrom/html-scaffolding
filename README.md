# HTML Scaffolding

Folder structure set up and workflow to develop HTML sites with gulp, tests, styleguide and documentaion.

```
$ npm install

$ gulp --all

```

Serve test report
```
$ gulp serveTest
```

Serve coverage report
```
$ gulp serveCoverage 
```

Serve docs
```
$ gulp serveDoc 
```

Serve css stat report
```
$ gulp serveStat 
```

Serve build
```
$ gulp serve 
```
